Investigación

HTML (HyperText Markup Language)

  • ¿Qué es? Es el lenguaje estándar conformado por etiquetas para crear la estructura de páginas web.

    • Características fundamentales:

  • Define la semántica y el contenido de una página (encabezados, párrafos, imágenes, enlaces, etc.).
  • Utiliza etiquetas para marcar elementos como párrafos y encabezados.
  • Es la base sobre la cual se construyen las páginas web.

CSS (Cascading Style Sheets)

  • ¿Qué es? Es un lenguaje de diseño utilizado para dar estilo y formato a las páginas web creadas con HTML.

    • Características fundamentales:

  • Define cómo se muestran los elementos HTML (colores, fuentes, tamaños, espaciado, etc.).
  • Permite separar la presentación de la estructura del contenido.
  • Utiliza selectores para aplicar estilos a elementos específicos.

JavaScript

  • ¿Qué es? Es un lenguaje de programación interpretado que se utiliza para crear páginas web interactivas.

    • Características fundamentales:

  • Permite manipular el DOM (Document Object Model) para cambiar el contenido y el estilo de una página dinámicamente
  • Se utiliza para crear animaciones, validar formularios, crear efectos especiales y mucho más.
  • Puede interactuar con el servidor a través de tecnologías como AJAX.

Etiquetas de estilo

  • ¿Qué son? Son elementos dentro del código HTML que permiten aplicar estilos CSS a elementos específicos.

    • Características fundamentales:

  • Se utilizan atributos como style para aplicar estilos directamente a un elemento.
  • Se pueden utilizar hojas de estilo externas para separar la presentación del contenido.
  • Los selectores CSS permiten aplicar estilos a múltiples elementos de forma eficiente.
